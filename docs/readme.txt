Програма предстовляет собой эмулятор интернет-магазина для 
покупки конструкторов Lego.
Програма обладает следующими основными функциями:
– ведение базы данных пользователей с возможностью их редактирования и удаления;
– ведение базы данных товаров, их категорий и корзин пользователей;
– имитация покупки товаров пользователями;
- регистрация пользователей;
- отображение пользовательского web-интерфейса.
Програма предстовляет собой клиент-серверную web-приложение.
Серверная часть функцианирует с использованием python 3.8.
Клиентская часть посторена с использованием Boostrap 4.
База данных SQLite.
